Welcome to Frame Docs

Frame is a tiny front-end framework that lets you quickly create flexible, well-structured projects. On these pages you will find anything you might need to build your project with Frame.

However, to get it to work, you'll need some working knowladge of SASS. So if you're not familiar with it, it's probably high time you did it!

Installation

Using a package manager is probably the best way of working with Frame.

npm

$ npm install frame-framework

If you're new to npm, you can read more about it at npmjs.com.

Just Download It

If you'd like to simply download the source code, you still can do it on Github.

First Steps

If you're not sure where to start you can use this template as a starting point: 

<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>Frame Template</title>

    <link href="style.css" rel="stylesheet">

    <!--[if lt IE 9]>
      <script src="https://oss.maxcdn.com/html5shiv/3.7.3/html5shiv.min.js"></script>
      <script src="https://oss.maxcdn.com/respond/1.4.2/respond.min.js"></script>
    <![endif]-->
  </head>
  <body>

    <!-- Your code goes here -->

  </body>
</html>

…where style.css is your CSS file, compiled from a SASS stylesheet with all the Frame goodies inside: 

/**** VARIABLES OVERRIDES ****/

Here you can customize Frame…


/**** FRAMEWORK ****/

@import "../frame/scss/core/components/label";
@import "../frame/scss/core/components/text";

/* NAVIGATION */
@import "../frame/scss/core/components/navbar";

/**** YOUR CSS ****/

FAQ

Let's clarify a few details.

Where's my single CSS file? Where's CDN?

Well, the simple answer is: nowhere. Frame just doesn't make sense this way.
If you're looking for a bloated, dumb, out-of-the-box solution you came to the wrong place.

Where's my Javascript?

Again, nowhere. No jQuery, no Vanilla, no Javascript. Trying to keep Frame as versatile as possible, implemention of your project's components is left up to you.
